## Configuration

To align your local environment with the expectations of **Final Draft Software** and film scripts management, consider the following configuration guidelines:

* **Environment Variables**: For related tooling, ensure paths to `.fdx` file directories are correctly set. While **Final Draft** is a standalone application, supporting scripts may reference `FDX_HOME` or similar project root variables.
* **File Structure**: It is recommended to maintain a flat folder hierarchy for active film scripts to prevent broken internal links within **Final Draft Software**.
* **File Extensions**: Always work with `.fdx` (Final Draft XML) or `.fdr` formats to ensure compatibility with **Final Draft** standards.
* **Backup Practices**: Configure automatic backups as standard practice for film scripts. **Final Draft Software** includes robust auto-save features that should remain enabled to protect creative content.
* **Template Setup**: Custom templates for sitcoms, dramas, and feature film scripts should be stored in the default **Final Draft** templates directory for quick access.
